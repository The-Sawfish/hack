Disclaimer & How to Use

This website provides Chrome bookmarklets for fun and learning. You should only use them on websites you are allowed to interact with. The bookmarklets run locally in your browser and do not affect the server.

To install a bookmarklet: open Chrome → go to any site → create a new bookmark → paste the JavaScript code from the box into the URL of the bookmark → save it. Then click the bookmark while on the target page.

⚠️ Bookmarklets may be blocked on some pages due to browser security. Works best in Chrome desktop.
