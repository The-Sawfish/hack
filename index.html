Important Disclaimer:

This site provides bookmarklets for Chrome only. All scripts run locally in your browser. They do not bypass security or give access to sites you are not allowed to use. Effects are temporary and reloading the page removes them. Use these tools only for learning, testing, or fun.

How to Install:

  1. Open Chrome browser.
  2. Create a new bookmark in the bookmarks bar.
  3. Copy the desired bookmarklet code from the box below.
  4. Paste it into the bookmark's URL field.
  5. Visit the target page and click the bookmarklet to run it.
